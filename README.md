## About Wellnot

Wellnot helps you track daily symptoms, moods, and tags. It's fully offline — all data stays on your device with no internet access required.

**Features:**
- Log symptoms with severity levels and mood
- Tag entries for easy categorization
- Calendar view for browsing your history
- Pin favorite symptoms and tags
- Daily reminder notifications
- Light and dark theme support
